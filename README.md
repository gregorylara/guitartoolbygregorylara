# GuitarTool

A free, all-in-one guitar practice suite built with vanilla JavaScript and the Web Audio API.

## Features

- 🎸 **Guitar Tuner** — Real-time pitch detection using the microphone
- 🎵 **Chord Library** — 40+ chord diagrams with SVG fretboard visualizations (Major, Minor, 7th, Sus, Add9, Maj7)
- ⏱️ **Metronome** — Precise Web Audio API scheduler with tap tempo, time signatures, and BPM presets
- 🎼 **Scale Explorer** — 14 scales across 5 style categories (Blues, Jazz, Modes, World) with fretboard diagrams

## Tech Stack

- Vite + vanilla JavaScript (ES modules)
- Web Audio API for tuner and metronome
- SVG-based fretboard diagrams (no canvas, no dependencies)

## Getting Started

```bash
npm install
npm run dev
```
